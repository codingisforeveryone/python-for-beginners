# Homework on classes

### Task 1:

Create a class `Rectangle` 
- The class should have the following attributes that can be set via the constructor:
    - width 
    - height
- Add a method area(): This method should calculate and return the area of the rectangle
- Add a method perimeter(): This method should calculate and return the perimeter of the rectangle
- Add a method info(): This method should print the rectangle's details, including the width, height, area, and perimeter.

### Task 2:

- Create two objects of the  class `Rectangle` with different width and height
- assign the area and perimenter of both methods into variables
- call the info method for both methods
- compute the sum of both areas and print it
